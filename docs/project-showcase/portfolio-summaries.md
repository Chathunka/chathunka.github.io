# Portfolio Project Summaries

These projects are recommended because they show system ownership, technical complexity, industrial relevance, or a strong working demonstration.

## 2D SLAM Mobile Robot

**Recruiter summary:** A mobile robot demonstration for real-time mapping, navigation visualization, and SLAM experimentation.

**Engineering value:** Demonstrates mapping and localization in autonomous systems, using 2D SLAM, Mobile robotics, LiDAR, Robot visualization.

**Resume-ready bullet:** Captured a working robot and live mapping workflow

## Beer Keg Telemetry Data Logger and RTU

**Recruiter summary:** A modular Wi-Fi/3G remote terminal that monitors beer flow, keg pressure, cooler and tap temperatures, and compressor power while supporting remote control.

**Engineering value:** Demonstrates industrial telemetry and embedded electronics in beverage dispensing, using GSM/GPRS, Wi-Fi, HTTP, MQTT, Azure IoT, Analog sensing, Pulse inputs.

**Resume-ready bullet:** Completed enclosure, electronics, sensor harness, user manual, and system flow documentation

## Industrial Dangerous Gas Monitoring System

**Recruiter summary:** A container-port monitoring solution for dangerous gases using field sensors, an industrial gateway, dashboards, alarms, and notifications.

**Engineering value:** Demonstrates industrial iot and safety monitoring in port and industrial safety, using Modbus, Industrial gas sensors, IIoT gateway, AWS, MERN, Alerts.

**Resume-ready bullet:** Deployed monitoring hardware and operational visualization in a port environment

## Four-Channel Humidifier Controller

**Recruiter summary:** An ATmega328P-based controller for independently driving four mist-maker channels.

**Engineering value:** Demonstrates firmware and pcb design in environmental control, using ATmega328P, Arduino, PCB design, Relay control, Power electronics.

**Resume-ready bullet:** Designed, fabricated, assembled, and demonstrated the controller hardware

## Modbus Oxygen Sensor Reader

**Recruiter summary:** A hardware and software test setup for reading oxygen sensor measurements over Modbus/RS485.

**Engineering value:** Demonstrates industrial communications in environmental monitoring, using Modbus RTU, RS485, MAX485, ESP32, O2 sensor.

**Resume-ready bullet:** Demonstrated live oxygen readings through multiple test configurations

## ET Robocon Entry Robot Control System

**Recruiter summary:** A LEGO EV3 competition controller combining PID line tracing, color-based decisions, precision turning, object carrying, and distance measurement.

**Engineering value:** Demonstrates autonomous control software in robotics competition, using LEGO EV3, PID control, Line following, Color sensing, Distance sensing.

**Resume-ready bullet:** Completed the simulated entry course with documented V-model design artifacts

**Recruiter summary:** A Python/OpenCV pipeline that converts photographed hand-drawn entity-relationship diagrams into structured relationships and SQL-oriented output.

**Engineering value:** Demonstrates computer vision and document understanding in developer tools and education, using Python, OpenCV, Grayscale processing, Denoising, Contour detection, Shape classification.

**Resume-ready bullet:** Produced a working research prototype and retained source code

## RFID Inventory and Swipe Reader Terminal

**Recruiter summary:** A packaged RFID reader terminal with display, wireless connectivity, and card/key-fob interaction.

**Engineering value:** Demonstrates embedded identification systems in inventory and identification, using RDM6300, RFID, ESP32, Arduino, LCD, Wi-Fi.

## GSM Smart Cooler Data Logger and Protection Unit

**Recruiter summary:** A cooler monitoring unit that measures temperature, vibration, noise, orientation, voltage, and current, reports through GSM/USB, and can protect the compressor during critical conditions.

**Engineering value:** Demonstrates embedded telemetry and condition monitoring in cold-chain and equipment monitoring, using GSM/GPRS, USB, HTTP, MQTT, IMU, Temperature sensing, Power monitoring, Relay control.

**Resume-ready bullet:** Completed electronics, enclosure, sensor unit, PC utility, demonstration, and user documentation

**Engineering value:** Demonstrates mechatronics and motion systems in sports technology, using DC motors, Motor speed control, Automatic feeder, Battery power, RF remote, Mechatronics.
