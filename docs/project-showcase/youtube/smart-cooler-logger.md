# GSM Smart Cooler Data Logger and Protection Unit

## SEO Title

GSM Smart Cooler Data Logger and Protection Unit | GSM/GPRS, USB, HTTP Engineering Project

## Description

This engineering demonstration presents gsm smart cooler data logger and protection unit. A cooler monitoring unit that measures temperature, vibration, noise, orientation, voltage, and current, reports through GSM/USB, and can protect the compressor during critical conditions.

Technologies used:
- GSM/GPRS
- USB
- HTTP
- MQTT
- IMU
- Temperature sensing
- Power monitoring
- Relay control

Engineering focus:
- Embedded telemetry and condition monitoring
- Cold-chain and equipment monitoring
- Prototype architecture, integration, and validation
