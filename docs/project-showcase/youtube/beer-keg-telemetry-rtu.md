# Beer Keg Telemetry Data Logger and RTU

## SEO Title

Beer Keg Telemetry Data Logger and RTU | GSM/GPRS, Wi-Fi, HTTP Engineering Project

## Description

This engineering demonstration presents beer keg telemetry data logger and rtu. A modular Wi-Fi/3G remote terminal that monitors beer flow, keg pressure, cooler and tap temperatures, and compressor power while supporting remote control.

Technologies used:
- GSM/GPRS
- Wi-Fi
- HTTP
- MQTT
- Azure IoT
- Analog sensing
- Pulse inputs

Engineering focus:
- Industrial telemetry and embedded electronics
- Beverage dispensing
- Prototype architecture, integration, and validation
