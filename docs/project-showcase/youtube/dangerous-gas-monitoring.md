## Description

This engineering demonstration presents industrial dangerous gas monitoring system. A container-port monitoring solution for dangerous gases using field sensors, an industrial gateway, dashboards, alarms, and notifications.

Technologies used:
- Modbus
- Industrial gas sensors
- IIoT gateway
- AWS
- MERN
- Alerts

Engineering focus:
- Industrial IoT and safety monitoring
- Port and industrial safety
- Prototype architecture, integration, and validation
